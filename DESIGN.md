# 思维阁 ThoughtCabinet - 设计文档

## 1. 产品概述

**产品名称**: 思维阁 (ThoughtCabinet)
**产品定位**: 主动认知操作系统 - 个人的知识管理 & 任务执行系统
**核心理念**: 港湾 / 安全屋 / 基地 - 放松、思考、行动

---

## 2. 设计理念

### 2.1 视觉风格

| 元素 | 描述 |
|------|------|
| **背景** | 天空氛围：深蓝渐变 + 柔和星光 + 漂浮云朵 |
| **主色调** | 晨曦金 (#e8c88a) - 温暖、指引 |
| **辅助色** | 晚霞紫 (#9b8abf)、晨曦蓝 (#7eb8da) |
| **字体** | SF Pro Display / PingFang SC |
| **圆角** | 10-20px，柔和友好 |
| **动画** | 轻微浮动、渐变过渡、星光闪烁 |

### 2.2 布局结构

```
┌──────────────────────────────────────────────────────────┐
│  侧边栏 (260px)     │        主内容区                     │
│  ┌──────────────┐  │  ┌────────────────────────────────┐ │
│  │ 🏠 思维阁     │  │  │ Topbar (搜索/语言/主题)        │ │
│  ├──────────────┤  │  ├────────────────────────────────┤ │
│  │ 导航          │  │  │                                │ │
│  │ • 总览        │  │  │      动态内容区域               │ │
│  │ • 待处理      │  │  │      (根据模块显示不同内容)      │ │
│  │ • 知识图谱    │  │  │                                │ │
│  ├──────────────┤  │  │                                │ │
│  │ 模块          │  │  │                                │ │
│  │ • 信息精炼厂  │  │  │                                │ │
│  │ • 熵减整理器  │  │  └────────────────────────────────┘ │

│  │ • 战术指挥室  │  │  [+ 快速添加按钮]                   │
│  │ • 认知蓝图    │  │                                     │
│  └──────────────┘  │                                     │
└──────────────────────────────────────────────────────────┘
```

### 2.3 功能模块

| 模块 | 英文名 | 核心功能 |
|------|--------|----------|
| **总览** | Dashboard | 系统状态、统计卡片、最近动态 |
| **待处理** | Inbox | 收集箱、待整理内容 |
| **信息精炼厂** | The Refinery | 阅读→笔记、30秒速读、AI对话 |
| **熵减整理器** | The Organizer | 自动分类、碎片合并、闪念胶囊 |
| **价值评估局** | The Evaluator | 四象限、毒舌VC、灵感碰撞 |
| **战术指挥室** | The Planner | 30分钟胶囊、技能阶梯、拖延热力 |
| **认知蓝图** | Knowledge Blueprint | 战争迷雾、理论连线板 |

## 4. 交互规范

### 4.1 导航

- 侧边栏固定宽度 260px
- 当前模块高亮显示（左边缘金色竖条）
- 悬停时有轻微右移动画

### 4.2 主题切换

- 右上角 🌓 按钮
- 支持 Light / Dark 模式
- 默认跟随系统
- 记住用户选择

### 4.3 语言切换

- 右上角 中/EN 切换
- 默认中文

### 4.4 响应式

- 移动端侧边栏可收起
- 最小宽度 375px

## 5. 技术实现

| 项目 | 技术 |
|------|------|
| 前端 | HTML5 + CSS3 + Vanilla JS |
| 部署 | Python http.server (单一端口) |
| 路由 | Hash-based (#/dashboard, #/refinery 等) |
| 主题 | CSS Variables + localStorage |
| 动画 | CSS transitions + keyframes |
