# 🏠 HOME - Thought Cabinet Prototype

**代号**: HOME  
**端口**: 60001  
**全称**: Hierarchical Organization & Management Environment

---

## 设计理念

HOME 是 Thought Cabinet 认知操作系统的核心入口，采用飞船指挥中心风格设计。

### 命名含义
- **H**ierarchical: 层级化的知识组织
- **O**rganization: 系统化的信息管理
- **M**anagement: 高效的任务调度
- **E**nvironment: 沉浸式的操作环境

### 视觉风格
- **主色调**: 深蓝/黑色背景 + 青色高亮
- **字体**: Inter + JetBrains Mono（科技感等宽）
- **特效**: 扫描线、发光边框、HUD元素
- **隐喻**: 飞船指挥舱 / 控制中心

---

## 功能模块

| 模块 | 代号 | 图标 | 功能描述 |
|------|------|------|----------|
| Dashboard | CMD | 🎯 | 全局态势感知，系统状态监控 |
| Refinery | REF | ⚗️ | 信息精炼厂，从信息到洞察 |
| Organizer | ORG | 🧩 | 熵减整理器，从混乱到秩序 |
| Evaluator | EVA | ⚖️ | 价值评估局，从想法到项目 |
| Blueprint | MAP | 🗺️ | 认知蓝图，知识图谱可视化 |
| Planner | PLN | 📋 | 战术指挥室，从计划到执行 |

---

## 技术栈

- **HTML5**: 单页应用架构
- **Tailwind CSS**: CDN引入，无需构建
- **原生 JavaScript**: 模块切换逻辑
- **Google Fonts**: Inter + JetBrains Mono

---

## 使用方式

1. 直接在浏览器中打开 `index.html`
2. 点击左侧侧边栏切换不同模块
3. 所有交互均为演示原型，无后端功能

- **SKY**: 云端同步 / 远程访问版本
