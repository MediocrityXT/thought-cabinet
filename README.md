  # 🌟 ThoughtCabinet: Active Cognitive OS
  ## Module 0. Dashboard (指挥舱) - 全局态势感知
  设计目标：用户进入系统第一眼看到的不是“静态列表”，而是“认知系统运转状态”和“最近进展“，“当前焦点”，“做点什么“


   * 核心功能：
       * 顶部 (Status Bar) 系统底层执行状况是否正常：
           * Vault 状态：显示当前连接的知识库路径。
           * 后台任务监视器：当 Analysis Task 运行时，显示动态进度条（当前处理文件/总进度），给用户“系统正在思考”的反馈。
           * 操作区：Scan (增量扫描) / Analyze (深度分析) / Export (数据导出)。
       * 核心指标卡 (Stats Cards) 一眼看清当前整体分布状况：
           * 展示 Total Notes, Pending Review (待评估想法), High Value (高价值项目), Cognitive Gaps (认知缺口) 四个关键数字。
       * 近期进展 (Recent Progress) 展示最近新提升的认知、新完成的任务：
           * 展示认知体系的补全路径、以及展示新完成的任务、统计任务数和影响力得分
       * 三列式动态流 (Activity Streams) 看见自己现在待做的任务：
           1. Review Queue (待办审批)：直接展示 Pending 状态的 Action，点击跳转 Evaluator。
           2. Focus Projects (重点项目)：展示 High Impact 且 Doing 的项目进度。
           3. Cognitive Alerts (认知警报)：展示系统发现的“知识断层”（Gap），引导去 Organizer 补充笔记。
       * 底部 (Quick Wins / Time Filler) 帮助减少启动困难：
           * "I have 30 mins" 模块：根据当前时间碎片，推荐 time_estimate <= 30 的任务。

  ## Module A: 信息精炼厂 (The Refinery) - From Info to Insight
  目标：解决“只收藏不阅读，只阅读不思考”的问题。建立阅读与笔记之间的强制转换机制。


   * 核心流程：
       1. 📥 投料口 (The Hopper)：
           * 支持粘贴 URL（公众号/博客/Paper）、文本块，或从“稍后阅读”列表导入。
           * AI 预处理：自动去除广告、提取正文，并生成一份 "30秒速读报告"（核心论点、关键数据、争议点）。
       2. 💬 交互式深挖 (Socratic Chat)：
           * 场景：读了一篇长文《Rust 内存安全详解》，云里雾里。
           * 功能：右侧是一个 AI 对话框。你可以问：“用大白话解释所有权机制”，或者 AI 反问你：“这篇文章提到的观点，和你之前读的 Go
             语言并发模型有什么区别？” —— 强迫你思考。
       3. 📝 结晶提取 (Crystallization)：
           * 关键交互：在对话过程中，AI 会把精彩的总结或你自己的感悟高亮。你点击“提取”，这些内容自动变为 Markdown 笔记。
           * 输出：一篇链接了原文的、属于你自己的永久笔记 (Permanent Note)。

  ## Module B: 熵减整理器 (The Organizer) - From Chaos to Order
  目标：全自动后台处理。你只管写/存，系统负责归位。


   * 功能：
       * 自动分类：基于语义将笔记归入 Domain。
        * 🧩 碎片拼接台：
           * AI 自动把过去一周零散的、只写了一句话的 5 个笔记抓出来，问你：“这 5 句话好像都在讲‘Web3 支付’，要不要合并成一篇文档？”
        * 🚮 想法 Tinder (Idea Swipe)：
           * 场景：扫描出 50 个未分类笔记/信息太少的笔记时，列表太枯燥。
           * 交互：卡片式界面。左滑“归档/垃圾”，右滑“值得发展”，上滑“立即加入待办”。快速清理积压。
       * 链接推荐：在精炼新笔记时，自动提示：“这篇笔记和你 3 个月前写的《分布式系统》观点冲突，要不要看看？”
       * ⚡ 闪念胶囊 (Thought Capsule)：
           * 一个始终悬浮的输入框（或快捷键唤起）。
           * 功能：不仅支持文字，支持“语音转文字”（通过 API）。
           * AI 预判：输入时 AI 实时打标签（是 Idea? Todo? Note? 还是纯 吐槽?）。AI 也会实时生成问题，帮助你进一步阐明想法，然后汇总整合成笔记。

  ## Module C: 价值评估局 (The Evaluator) - From Idea to Project
  目标：把“想法”变成“项目”，或果断丢弃。

   * 创意功能点：
       * ⚖️ 毒舌 VC (The Roasting VC)：
           * 场景：你有了一个新 App 的点子。
           * 交互：点击“评估”，AI 扮演一个刻薄的风险投资人。
           * 输出：生成一份 SWOT 分析，并给出残酷的评语（“这只是个 Feature，不是 Product”），逼你想清楚商业模式。
       * 📈 价值四象限 (Impact/Feasibility Matrix)：
           * 可视化：一个动态散点图。X轴=可行性，Y轴=影响力。
           * 交互：你的所有 Project 散落在图上。
           * 决策：
               * 右上角（高优）：Do Now。
               * 左上角（难但有价值）：Breakdown (拆解)。
               * 右下角（容易但没用）：Delegate/Delete。
       * 💡 灵感碰撞机 (Serendipity Engine)：
           * 功能：每天随机抽取两个不同领域的笔记（例如“React Hooks”和“建筑设计模式”），强行问
             AI：“它们之间有什么共同的哲学？”——帮助你产生跨界的新理论。
       * 决策漏斗：Idea -> 评估 -> 归档/立项。


  ## Module D: 认知蓝图 (The Knowledge Blueprint)
  目标：通过视觉化让你看到自己的“认知缺口”。


   * 创意功能点：
       * 🗺️ 战争迷雾 (Fog of War)：
           * 可视化：在知识图谱上，你已有的笔记是亮色的。你没有笔记，但该领域通常包含的概念（AI 基于知识库补充）是灰色的、被迷雾覆盖的。
           * 激励：你一眼就能看到“我虽然懂 React，但‘性能优化’这块区域全是迷雾”，激发探索欲。
       * 🕸️ 理论连线板 (Detective Wall)：
           * 交互：类似侦探电影里的线索墙。允许你手动把几个笔记钉在墙上，用红线连起来，并写上你的推论（生成的理论）。AI
             可以辅助推荐“嫌疑人”（相关笔记）。


  ## Module E: 战术指挥室 (The Planner) - From Plan to Done
  目标：执行。解决“启动困难症”。


   * 创意功能点：
       * ⏱️ 30分钟胶囊 (The 30-min Pill)：
           * 场景：我好累，只有 30 分钟，不想思考做什么。
           * 交互：点击按钮“我有 30 分钟”。
           * 算法：AI 从所有 Project 中筛选出 Context 完整、难度低、预估耗时短 的原子任务。
           * 结果：直接把 IDE 命令或阅读链接甩在你脸上：“别废话，读这篇文档的第 3 章节”。
       * 🪜 阶梯生成器 (Ladder Builder)：
           * 场景：左上角那些“难但有价值”的项目（如“写一个操作系统”），让人望而生畏。
           * 功能：AI 自动将这个庞然大物拆解为 50 个微小步骤。
           * 展示：游戏化的技能树。你只需要点亮第一个图标：“下载 GCC 编译器”。
       * 📉 拖延症热力图：
           * 分析你哪些领域的 Action 长期滞留。告诉你：“你总是在‘理财’相关的任务上拖延，是不是该放弃这个领域或者换个方法？”
